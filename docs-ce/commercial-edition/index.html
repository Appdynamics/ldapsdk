Additional Functionality in the Commercial Edition

The Commercial Edition of the UnboundID LDAP SDK for Java provides all of the capabilities available in the Standard Edition, but also adds additional functionality that may only be used in conjunction with the Ping Identity, UnboundID, and Alcatel-Lucent 8661 Directory Server and therefore is not applicable to other directory servers.
